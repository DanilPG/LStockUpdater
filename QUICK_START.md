## 🎯 Следующие шаги

### 1. Локальный тест (опционально)

```bash
cd flask_app

# Создайте виртуальное окружение
python3 -m venv venv

# Активируйте его
source venv/bin/activate  # Mac/Linux
# или
venv\Scripts\activate  # Windows

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
python3 app.py
```

Приложение будет доступно по адресу: http://localhost:5000

**ВАЖНО:** При первом запуске будет создан пользователь `admin` со случайным паролем. Пароль будет выведен в консоль. Сохраните его!

### 2. Подготовка к деплою

#### Создайте иконки (опционально)

```bash
# Если установлен Pillow
```

```bash

```

### 3. Деплой на бесплатный хостинг

#### Вариант 1: Render.com (Рекомендуется)

1. Зарегистрируйтесь на https://render.com через GitHub
2. Создайте новый репозиторий на GitHub и загрузите код
3. В Render создайте "New Web Service"
4. Подключите ваш GitHub репозиторий
5. Настройте:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn app:app`
6. Добавьте переменную окружения `FLASK_SECRET_KEY` (случайная строка)
7. Нажмите "Create Web Service"
8. Дождитесь деплоя (5-10 минут)
9. Откройте Logs и найдите пароль администратора

#### Вариант 2: Railway.app

1. Зарегистрируйтесь на https://railway.app
2. Создайте новый проект из GitHub репозитория
3. Railway автоматически определит Python проект
4. Добавьте переменные окружения:
   - `FLASK_SECRET_KEY` (случайная строка)
   - `PORT`: `5000`
5. Нажмите "Deploy"

#### Вариант 3: PythonAnywhere

1. Зарегистрируйтесь на https://www.pythonanywhere.com
2. Создайте бесплатный аккаунт "Beginner"
3. Создайте новое Web App (Flask)
4. Загрузите файлы через консоль или Git
5. Настройте WSGI файл (см. pythonanywhere.py)
6. Установите зависимости: `pip install -r requirements.txt`
7. Добавьте переменную окружения `FLASK_SECRET_KEY`
8. Перезагрузите приложение

### 4. Установка на телефон

После успешного деплоя:

#### iOS (iPhone/iPad)
1. Откройте приложение в Safari
2. Нажмите "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите "На экран «Домой»"
4. Нажмите "Добавить"

#### Android
1. Откройте приложение в Chrome
2. Нажмите на меню (три точки)
3. Выберите "Добавить на главный экран"
4. Нажмите "Добавить"

## 🔐 Первые действия после входа

1. **Смените пароль** на сложный (минимум 12 символов)
   - Перейдите в "Настройки"
   - Введите текущий и новый пароль
   - Нажмите "Изменить пароль"

2. **Проверьте работу**
   - Выберите маркетплейс
   - Отметьте магазин
   - Нажмите "Обновить остатки"
   - Проверьте результат

## 🆘 Проблемы?

### Приложение не запускается локально

```bash
# Проверьте Python версию
python3 --version  # Должна быть 3.11+

# Переустановите зависимости
pip install -r requirements.txt --upgrade
```

### Ошибка импорта модулей

Убедитесь, что вы находитесь в корневой директории проекта:

```bash
cd /Users/danil/Desktop/StockApp
python3 flask_app/app.py
```

### Проблемы с деплоем

1. Проверьте логи на хостинге
2. Убедитесь, что все зависимости установлены
3. Проверьте переменные окружения
4. Убедитесь, что порт правильный (обычно 5000)
