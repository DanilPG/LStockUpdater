# 📦 LStock Updater - Flask Web App

Мобильное веб-приложение для управления остатками на маркетплейсах (Ozon, Wildberries, Яндекс.Маркет).

## ✨ Возможности

- 📱 **Адаптивный мобильный интерфейс** - работает как на телефоне, так и на компьютере
- 🔐 **Безопасная аутентификация** - вход по логину/паролю с хешированием
- 🛒 **Управление остатками** - обновление и обнуление для всех маркетплейсов
- 📊 **Статистика и история** - отслеживание всех операций
- 🔄 **PWA поддержка** - можно установить на телефон как приложение
- 🌐 **Бесплатный хостинг** - готов к деплою на Render, Railway, PythonAnywhere

## 🚀 Быстрый старт

### Локальный запуск

```bash
# Клонируйте репозиторий
git clone <repository-url>
cd StockApp/flask_app

# Создайте виртуальное окружение
python -m venv venv

# Активируйте его
# Mac/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt

# Сгенерируйте иконки (опционально)
pip install Pillow
python generate_icons.py

# Запустите приложение
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

### Первый вход

При первом запуске будет создан пользователь `admin` со случайным паролем. Пароль будет выведен в консоль:

```
============================================================
ВНИМАНИЕ! Создан пользователь admin с паролем:
<случайный-пароль>
Срочно измените пароль после первого входа!
============================================================
```

## 📱 Установка на телефон

После деплоя на хостинг:

### iOS (iPhone/iPad)
1. Откройте приложение в Safari
2. Нажмите "Поделиться" (квадрат со стрелкой вверх)
3. Выберите "На экран «Домой»"
4. Нажмите "Добавить"

### Android
1. Откройте приложение в Chrome
2. Нажмите на меню (три точки)
3. Выберите "Добавить на главный экран"
4. Нажмите "Добавить"

   - $5 кредитов/месяц бесплатно

## 🛡️ Безопасность

Приложение включает:

- ✅ Хеширование паролей (bcrypt)
- ✅ Таймаут сессии (1 час)
- ✅ HTTPS шифрование (на хостинге)
- ✅ CSRF защита
- ✅ Логирование всех действий

### Рекомендации:

- Используйте сложный пароль (минимум 12 символов)
- Регулярно меняйте пароль
- Не передавайте учетные данные третьим лицам
- Используйте VPN при работе с публичных сетей

## 🔧 API

### Аутентификация

Все API endpoints требуют авторизации.

### Endpoints

- `POST /api/update` - Обновление остатков
- `POST /api/reset` - Обнуление остатков
- `GET /api/stores` - Список магазинов
- `GET /api/history` - История операций
- `GET /api/stats` - Статистика
- `POST /api/clear-history` - Очистка истории
- `POST /api/change-password` - Смена пароля
- `GET /api/session-info` - Информация о сессии

## 📊 Функции

### Обновление остатков

1. Выберите маркетплейс (Ozon, Wildberries, Яндекс)
2. Отметьте нужные магазины
3. Нажмите "Обновить остатки"
4. Следите за результатом в выводе

### Обнуление остатков

1. Выберите маркетплейс
2. Отметьте магазины для обнуления

4. Подтвердите действие

- Фильтрация по дате
- Очистка истории

### Настройки

- Смена пароля
- Просмотр информации о сессии
- Информация о безопасности

## 🐛 Troubleshooting

### Приложение не запускается

```bash
# Проверьте зависимости
pip install -r requirements.txt

# Проверьте Python версию
python --version  # Должна быть 3.11+
```

### Ошибка импорта модулей

Убедитесь, что вы находитесь в корневой директории проекта:

```bash
cd StockApp  # Вернитесь в корень проекта
python flask_app/app.py
```

### Проблемы с PWA

- Убедитесь, что HTTPS включен
- Проверьте manifest.json
- Очистите кэш браузера
