# 📦 LStock Updater - Flask Web App - Обзор проекта

## ✨ Что создано

Полнофункциональное Flask веб-приложение с мобильным интерфейсом для управления остатками на маркетплейсах.

## 📁 Структура проекта

```
flask_app/
├── app.py                      # Основное приложение Flask (400+ строк)
├── requirements.txt            # Зависимости Python
├── .gitignore                  # Git ignore файл
│
├── templates/                  # HTML шаблоны
│   ├── base.html              # Базовый шаблон с CSS и JS
│   ├── login.html             # Страница входа
│   ├── dashboard.html         # Главная панель управления
│   ├── settings.html          # Страница настроек
│   └── error.html             # Страница ошибки
│
├── static/                     # Статические файлы
│   ├── manifest.json          # PWA манифест
│   └── sw.js                  # Service Worker для PWA
│
├── render.yaml                 # Конфигурация для Render.com
├── railway.json               # Конфигурация для Railway.app
├── pythonanywhere.py          # Конфигурация для PythonAnywhere
│
├── generate_icons.py          # Генератор иконок (требует Pillow)
├── create_simple_icons.py     # Простой генератор placeholder иконок
│
├── README.md                  # Документация проекта
├── DEPLOYMENT.md              # Подробная инструкция по деплою
├── QUICK_START.md             # Быстрый старт
└── PROJECT_OVERVIEW.md        # Этот файл
```

## 🔧 Технологии

### Backend
- **Flask 3.0.0** - веб-фреймворк
- **Werkzeug 3.0.1** - WSGI тулкит
- **Gunicorn 21.2.0** - WSGI сервер для продакшена
- **python-dotenv 1.0.0** - управление переменными окружения

### Frontend
- **HTML5** - разметка
- **CSS3** - стили (встроенные в base.html)
- **JavaScript (ES6+)** - интерактивность
- **Fetch API** - AJAX запросы

## 🛡️ Меры безопасности

### Реализованные
1. **Аутентификация**
   - Логин/пароль
   - Хеширование паролей
   - Таймаут сессии

2. **Защита данных**
   - HTTPS (на хостинге)
   - Валидация входных данных
   - CSRF защита (готова)

3. **Логирование**
   - Все действия записываются
   - Отслеживание неудачных попыток

### Рекомендации для пользователя
- Использовать сложный пароль (минимум 12 символов)
- Регулярно менять пароль
- Не передавать учетные данные
- Использовать VPN в публичных сетях

## 📊 API Endpoints

### Публичные
- `GET /` - перенаправление на дашборд или логин
- `GET /login` - страница входа
- `POST /login` - обработка входа

### Требующие авторизации
- `GET /dashboard` - главная панель
- `GET /settings` - настройки
- `GET /logout` - выход

### API
- `GET /api/stores` - список магазинов
- `POST /api/update` - обновление остатков
- `POST /api/reset` - обнуление остатков
- `GET /api/history` - история операций
- `GET /api/stats` - статистика
- `POST /api/clear-history` - очистка истории
- `POST /api/change-password` - смена пароля
- `GET /api/session-info` - информация о сессии
